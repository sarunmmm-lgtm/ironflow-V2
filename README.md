# IRONFLOW AI v2 Production Starter

เวอร์ชันเริ่มใหม่ของ IRONFLOW v2 สำหรับเอาขึ้น GitHub/Vercel

## Run locally

```bash
npm install
npm run dev
```

เปิด `http://localhost:3000`

## Pages

- `/` Home
- `/train` Workout Session
- `/ai` AI Insight
- `/stats` Statistics
- `/library` Exercise Library
- `/exercise/[id]` Exercise Detail
- `/profile` Profile
- `/settings` Settings
